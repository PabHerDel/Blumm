Colocar las imágenes del sitio en esta carpeta.

Sugerencia de nombres (opcional):
  - hero.jpg             → imagen de fondo del home
  - education.jpg        → fotografía sticky de education.html
  - work-01.jpg          → primera imagen de la galería
  - work-02.jpg ...

Formatos recomendados: .jpg para fotos (peso), .webp si querés aún
menos peso. Resolución sugerida para el hero: 2400px de ancho mínimo.

Cómo se referencian en el HTML:
  <img src="img/hero.jpg" alt="...">
